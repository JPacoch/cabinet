# Contributing to cabinet

First off, thank you for entering the "engine room" of my thoughts. This repository houses both the code that powers the site and the drafts of my articles. 

---

## Content & philosophy

The essays found in the `/texts` directory are my personal reflections.

### 1. Typos & Grammar
If you spot a typo, a dangling participle, or a broken link, please **open a pull request**. I value clarity, and I’m grateful for the extra set of eyes.

### 2. Philosophical Disagreements
I love a good argument, but a pull request is a poor place for a discussion.
* **Don't** open a PR to rewrite my conclusions or logic.
* **Do** reach out on my socials or directly via email.

### 3. Intellectual Property
Please respect the license. You are encouraged to quote these works with proper attribution. However, please do not republish full essays without prior consent.

##  The Process

1.  **Open an issue:** For anything larger than a typo, please open an issue first.
2.  **Project board:** Check the **Projects** tab to see what I’m currently working on.
3.  **Branching:** Please create a feature branch for your changes (e.g., `fix/header-alignment` or `bugfix/typo-in-determinism-essay`).
